Motherboard

The motherboard is a printed circuit that allows communication between the different parts of the computer. The RAM, CPU, and GPU attach directly to the motherboard, while the motherboard is connected to power supply. One of the main jobs of the motherboard is to hold the CPU and use two "bridges", the chipset, to connect the CPU to the different parts of the computer. The northbridge gives the CPU fast access to the memory and the slightly slower southbridge connects the CPU to the USB ports, PCI bus, and the harddisk or IDE connections. The motherboard has serveral ports and slots to allow things to be plugged in. PCI connections for sound and video capture cards and network cards. Most motherboards will also have USB 3.0 and 2.0 for external connections, memory slots to hold the RAM, IDE slots for hard drives, and AGP for the GPU. The motherboard is the basis of the computer, connecting all the different components together.

CPU

CPU, or Central Processing Unit, does most of the hard work. The CPU does all of the calculations for the computer, and the computer's speed is normally based around how fast the processer is. On the bottom of the CPU, or where it attaches to the motherboard, there are small metal pins. These pins have to match up with the correct pins on the mother board, or the Computer will not work. The first computers all used a type of PGA, Pin Grid Array, known as Socket 7. However, in the modern era, all most computer use different Sockets, or uses a LGA, Land Grid Array, where the pins are part of the socket instead of the CPU. It is always important to select the correct motherboard for your CPU.See motherboard for more information on the Chipset.

RAM

RAM, not to be confused with storage, is the memory on the computer. RAM stands for Random Access Memory because you can get any piece of information out of it, as long as you have the row and collumn of the information is stored in, a bit like having an address for a house you want to go to. All of the data in the memory is stored in cells as binary. Each cell holds a one or zero, depending on whether or not the cell has electrons in it. There are thousands of these cells; it might only take 8 cells to store one letter, but it takes a lot more to store anything of value. For example, many computers have 16GB of RAM, or 16,000,000,000 cells. The main point of the RAM is to be the workspace for the CPU, acting as short-term memory for data and information that will either be used shortly for processing, or has been put there to be used in other open applications
